title: About this site ## Open APC The [OpenAPC initiative](https://openapc.net) releases data sets on fees paid for open access journal articles by universities and research institutions under an open database license. OpenAPC is operated by [Bielefeld University Library](http://www.ub.uni-bielefeld.de/english/). ## Data Access Main place for OpenAPC to collect and maintain its data is [GitHub](https://github.com/OpenAPC), where all core data files are kept and redacted in CSV format. This site and its open [backend](https://olap.openapc.net) were established to improve accessibility and re-use of the data, which means that there are 3 ways to access the OpenAPC data sets, differing in their tradeoff between flexibility and ease of handling: * Raw data The most basic way to access the Open APC data is the [main APC file](https://raw.githubusercontent.com/OpenAPC/openapc-de/master/data/apc_de.csv) on GitHub. It is a [CSV](https://en.wikipedia.org/wiki/Comma-separated_values) file, meaning that it should be easily processable by a wide range of tools and programming languages. A schema description of the file can be found [here](https://github.com/OpenAPC/openapc-de/wiki/schema). * OLAP Since the raw CSV data can be difficult to analyze without tool support, a more elegant method is querying the OpenAPC [OLAP Server](https://olap.openapc.net). This service implements an [online analytical processing](https://en.wikipedia.org/wiki/Online_analytical_processing) framework based on [cubes](http://cubes.databrewery.org/), forming the backend of treemaps.openapc.net. For usage instructions have a look at this [HOWTO](https://github.com/OpenAPC/openapc-olap/blob/master/HOWTO.md). * Treemaps The treemaps on this site present the most easy and intutitive way to browse and inspect the OpenAPC data. It is possible to either view the data for single institutions or the [whole data set](/apcdata/openapc/). ## Acknowledgements This site is based on [OffenerHaushalt](http://offenerhaushalt.de), a project by [the Open Knowledge Foundation Germany](http://okfn.de) aimed at making German budget data transparently and openly accessible.
